
# GreenLedger - Farmer-to-Consumer Agricultural Platform

GreenLedger is a modern web application designed to connect farmers directly with consumers while ensuring transparent and efficient agricultural product tracking. The platform features modules for farmers, warehouses, and consumers, with real-time tracking and quality assurance.

- **Farmer Module**: Crop management, expense tracking, and QR code generation for products
- **Warehouse Module**: Inventory management and product movement tracking
- **Consumer Module**: Live product tracking, ordering system, and quality verification
- **Uzhavan Santhai Hub**: Direct marketplace connecting farmers with consumers
- **Multi-language Support**: Built-in language selector for accessibility
- **Real-time Notifications**: Product movement and status updates
- **Carbon Footprint Tracking**: Environmental impact monitoring
- **AI-powered Crop Analysis**: Smart farming recommendations
